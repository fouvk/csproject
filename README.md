## Introduction/Overview

A small arcade like game project made for a grade 12 computer science project in ~ 1 week

## Project Info

You are first prompted to enter a username to play the game (which will be used to store highscores).
You play as a small red circle being swarmed by the "void" (black circles of various sizes/speeds) from the top of the screen, colliding off of them and being pushed around. You must move around using the arrow keys/WASD to avoid being pushed off the screen (with increasing difficulty/score gains), where its game over and your score is saved.

## How to run

Download the .java files in /src and run the main method in Game.java, or just download the whole thing to vscode.

## Project Goals

To create a conceptually simple but addicting game with dynamic gameplay based on randomness, movement, on the spot planning, and visual identification (once the circles start overlapping more it becomes more difficult, needing more attention).
Intuitive and very smooth controls with correct normalization, contrasting and clear visuals, lightweight and consistent/expected collision behavior, and a hyperbolic decay function used for difficulty scaling to ramp up quicker in the beginning for engagement, but slows down over time to avoid huge and sudden difficulty spikes.
